# AI 人格市场 - 企业级AI人格交易平台

## 项目简介

AI 人格市场是一个面向企业级客户的 AI 人格交易与服务平台，致力于帮助企业快速找到适合客服、销售陪练和培训讲师场景的 AI 人格解决方案。

## 核心功能

### 1. 企业首页
- 实时数据概览（调用量、活跃人格、满意度）
- 推荐人格展示
- 快捷操作入口
- 使用趋势图表

### 2. 人格库
- 多维度筛选（行业、任务类型、合规等级、响应风格）
- 人格卡片展示
- 人格详情弹窗
- 加入候选清单功能

### 3. 评测对比
- 选择多个 AI 人格进行对比
- 输入问题测试不同人格的回答
- 回答质量评分（准确度、专业度、友好度）
- 可视化对比结果

### 4. 候选清单
- 查看已收藏人格
- 提交采购申请
- 填写用途说明和期望日期
- 审批状态跟踪

### 5. 使用监控
- 实时数据大屏
- 员工使用明细
- 异常告警列表
- 满意度趋势

### 6. 管理后台
- 采购审批管理
- 企业额度配置
- 告警处理
- 数据统计分析

## 技术栈

### 前端
- **React 18** + TypeScript
- **Vite** - 快速构建工具
- **Tailwind CSS** - 原子化 CSS 框架
- **React Router v6** - 路由管理
- **Zustand** - 状态管理
- **Recharts** - 数据可视化
- **Lucide React** - 图标库

### 后端
- **Express.js** - RESTful API
- **TypeScript** - 类型安全
- **CORS** - 跨域支持

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
# 启动前端开发服务器
npm run client:dev

# 启动后端 API 服务器
npm run server:dev

# 同时启动前后端
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 代码检查

```bash
npm run check
```

## 项目结构

```

├── src/
│   ├── components/        # React 组件
│   │   ├── common/       # 通用组件（Button, Card, Modal等）

│   │   ├── personality/   # 人格相关组件
```

## 主要页面路由

| 路由 | 页面名称 | 描述 |
|------|----------|------|
| `/` | 首页 | 企业首页概览 |
| `/personalities` | 人格库 | 浏览和筛选 AI 人格 |
| `/compare` | 评测对比 | 多人格对比测试 |
| `/shortlist` | 候选清单 | 管理收藏人格和采购申请 |
| `/monitor` | 使用监控 | 查看使用数据和告警 |
| `/admin` | 管理后台 | 管理员功能 |

## 开发指南

### 添加新组件
1. 在 `src/components/` 下创建组件文件
2. 使用通用组件库（Button, Card 等）保持一致性
3. 遵循现有的样式规范

### 添加新页面
1. 在 `src/pages/` 创建页面组件
2. 使用 `PageContainer` 包裹内容
3. 在 `App.tsx` 添加路由

### 添加新状态
1. 在 `src/stores/` 创建 Zustand store
2. 在需要的地方导入使用
